# Todo API

a todo api

## how to run

```
npm install
npm start
```

## Endpoints

List : GET /todos
Create : POST /todos
Get (Id) : GET /todos/:id
Update (Id) : PUT /todos/:id
Delete (Id) : DEL /todos/:id
